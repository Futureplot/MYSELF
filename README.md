# what have i even done?

So, im just a crazy indie developer. Ive been making games, if you would call them that, since 2020-2021. I started off in gdevelop and roblox studio, and before that i used scratch in mostly non-game stuff.

### when does it get better?

Well, you see, i started to step away from the simpler engines in order to create big projects with big ambitions.  I was inexperienced as hell. I started using Gdevelop, Ursina Engine,  Unreal Engine 4.2x, and rarely unity. I enjoyed the process a lot. 

During late 2022, i stumbled upon a group of people ambitious to make a horror game in the newly introduced 3D capabilities of Gdevelop. I was a part of the founding people, and together we made a game titled No Way Out. It was a great process, and taught me a lot. Unfortunately, the game development was stopped during development... Im sure you can find stuff about it if you dig deep enough on the internet. 


Then, i started to make my own games, all by myself.
